使用IDE打开Vue2项目

可以看到Vue原生支持四条npm命令：

  1. npm install

    npm作为包管理器，在Vue2项目中根目录下的package.json文件记录项目中所需要的所有包、版本、配置信息，install的所有包在node_modules文件夹下。

    创建Vue2项目时，默认的package.json文件已经配置好了基本所需的包，而且已经install过了，因此可以直接启动项目，不需要再跑一遍npm install。

    如果是从别的地方copy或者clone来的vue项目，因为node_modules文件夹体积过大的原因，基本上copy或者clone的Vue项目中是不会含有node_modules文件夹的，这个时候就需要在cmd中运行npm install命令，下载package.json文件中的包生成node_modules文件夹。

  2. npm run serve

    运行Vue项目的命令，等同于vue-cli-service serve，可以在package.json文件中看到：

    npm脚本名称可以由服务所有者自定义，在这里”serve”改成自定义名称例如”dev”，那么就运行npm run dev，最终执行的都是相同的命令”npm run vue-cli-service serve”。

    尝试启动服务，打开终端，在项目的路径下执行npm run serve：

    项目正常启动，浏览器打开就是Vue2的初始默认页面：

    停止服务在终端用CTRL+C即可。

  3. npm run build

    打包命令，执行后生成的dist文件夹（默认dist，可以在配置中修改名称）中，运行index.html页面就可以像打开静态网页一样打开Vue的项目。

  4. npm run lint

    静态分析所有在 src/ 和 test/ 下面的 JS 以及 Vue 组件文件。
